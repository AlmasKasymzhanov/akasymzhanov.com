## 5.1 Полный путь клиента

```
INSTAGRAM (150K) → ЧЕЛЛЕНДЖ (бесплатный, 30 дней) → QUIZ-ВОРОНКА ("Какой у тебя тип фигуры?")
    ↓                                                                ↓
TELEGRAM (community)                                         ПЕРСОНАЛЬНЫЙ ПЛАН
    ↓                                                                ↓
ТОВАР (резинки/леггинсы, 6-12K₸)                            APP-ПОДПИСКА (2500₸/мес)
    ↓ (QR-код в коробке → 1 мес бесплатно)                         ↓
APP → COMMUNITY → UPSELL (спортпит, VIP, BodyPro) → RETENTION (66-дневный челлендж)
```

## 5.2 Почему связка даёт x3-5 LTV

| Сценарий | Средний LTV клиента | Объяснение |
|----------|---------------------|------------|
| **A: Только товары** | ₸8 000 | Одна покупка резинок/леггинсов. Нет повторных касаний |
| **B: Товары + App** | ₸28 800 | Товар ₸8000 + подписка ₸1800/мес × 8 мес = ₸14 400 + cross-sell ₸6 400 |
| **C: Полный Phygital** | ₸42 000+ | Товар + подписка + марафоны + спортпит + BodyPro скидки. Retention 12+ мес |

**Коэффициент LTV:** Phygital (C) даёт **x5.25 LTV** по сравнению с «только товары» (A).

**Доказательства из мировых кейсов:**
- BetterMe: 90%+ выручки — подписки, не товары. Маржа софта ~85% vs товаров ~40%

- POPFLEX: дизайн-видео на YouTube → прямые продажи. Товар и контент неразделимы

# 10. ПРИЛОЖЕНИЕ — ТЕХНИЧЕСКИЙ СТЕК

## 10.1 MVP: рекомендация

| Параметр | Решение | Обоснование |
|----------|---------|-------------|
| Платформа | **Flutter** (кроссплатформа iOS + Android) | Один код → 2 платформы. Экономия ~45% vs натив. MVP за 2-3 мес |
| Backend / BaaS | **Supabase** (PostgreSQL + Auth + Storage + Edge Functions) | Не нужен отдельный backend-разработчик на старте. Бесплатный план до 50K MAU |
| Хостинг | **Vercel** (для веб-части, quiz-воронки) | Автоскейлинг, CDN, бесплатный старт |
| Стоимость MVP | **₸7-12M ($14-24K)** | Flutter MVP за 2-4 месяца. Казахстанские/узбекские разработчики |
| Альтернатива: No-code MVP | **Telegram Web App (TWA)** | Минимальный барьер входа. Запуск за 2-4 недели. ₸500K-1M |
| Quiz-воронка | **Typeform/Tally → Zapier → App** | Быстрый запуск, A/B тестирование вопросов |
| Telegram-бот | **Telegraf.js + Supabase** | Трекинг челленджа, напоминания, streak |
| Платежи app | **Apple IAP + Google Play Billing + Kaspi** | Стандартные app-store платежи + локальный Kaspi |
| Аналитика | **Amplitude** (бесплатно до 50K MAU) | User behavior, retention, funnels |

## 10.2 Рекомендация: No-code → Custom

**Фаза 0-2 (месяцы 1-3):** Telegram Web App + бот для трекинга. Стоимость ₸500K-1M. Позволяет валидировать спрос до разработки полного приложения.

**Фаза 2-3 (месяцы 3-7):** Flutter MVP. Стоимость ₸7-12M. Переносим validated аудиторию из TG в полноценное приложение.

**Фаза 4+ (месяц 12+):** Полноценная платформа с AI, wearables, B2B. Стоимость ₸15-25M.

## 10.3 Контент: адаптация под СНГ-кухню

- Планы питания включают: плов (расчёт КБЖУ на порцию), бешбармак, лагман, шубат, курт, баурсаки

## B. Глоссарий

- **Phygital** — Physical + Digital: бизнес-модель, где физические товары и цифровые продукты работают как единая экосистема
- **LTV** — Lifetime Value: пожизненная ценность клиента (сколько денег он принесёт за всё время)
- **CAC** — Customer Acquisition Cost: стоимость привлечения одного платящего клиента
- **ARPU** — Average Revenue Per User: средний доход на одного пользователя
- **MAU/DAU** — Monthly/Daily Active Users
- **MRR** — Monthly Recurring Revenue: ежемесячный рекуррентный доход
- **Churn** — процент пользователей, отменяющих подписку за месяц
- **TWA** — Telegram Web App: мини-приложение внутри Telegram
