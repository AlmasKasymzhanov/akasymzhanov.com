## Привет!

Меня зовут Алмас Касымжанов. Я занимаюсь аналитикой маркетплейсов — Kaspi.kz, Wildberries, и других площадок.

На этом сайте вы найдёте:

- **Инструменты** — WB Niche Analyzer для анализа ниш на Wildberries, гайд по MPStats API
- **Отчёты** — enterprise-аналитика для бизнеса (пример: анализ рынка очков для BG Optic)
- **Блог** — заметки об аналитике, данных и маркетплейсах

### WB Niche Analyzer
